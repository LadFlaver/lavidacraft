What is LavidaCraft?

LavidaCraft is a resource pack for Minecraft made by Lad_Flaver. It is not a retexture of the game, but rather improvements and tweaks to various parts of Minecraft while striving to preserve the Vanilla feel. These changes may range from an entire redo, to the smallest change.

Features:

New Sun and Moon
Changes to a Few Blocks
New Bottle and Potions
Changes to Netherite
Tweaks to the HUD
Animated redstone and torches*
Changes to a Few Items
Less Intense Ore Shading
Better Potted Cactus
Fixes Seagrass Clipping

Other features include:

Lad's Rail Fix, minor touch ups to some mobs, new chorus flower texture and model, an alternate enchantment effect, minor consistency changes, and a bit more! Click here for a list of all features and which game versions they are supported on.

*The animations are an approximation of what the content looks like in the game. Due to software limitations, the previews shown on this page may differ in smoothness or speed from those in the game.
