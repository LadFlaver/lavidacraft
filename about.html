What is LavidaCraft?

LavidaCraft is a resource pack for Minecraft made by me (Lad_Flaver). It is not a retexture of the game, but rather improvements and tweaks to various parts of Minecraft while striving to preserve the Vanilla feel.

Why do some features differ between the Java and Bedrock versions of LavidaCraft?

Minecraft Java Edition and Minecraft Bedrock Edition implement many features very differently from each other. This makes it difficult to make features consistent between versions. There are some features which simply aren't available or too difficult for me to implement into the Bedrock version. I will continue to do my best to support the Bedrock version. When and how did LavidaCraft start?

LavidaCraft started early in 2019 and was only shared in a Discord server with my friends. The name of the pack is based off my username at the time, "Laviday". There's no record of when I made the first release of LavidaCraft, but the second release was made on March 28, 2019. LavidaCraft originally started because there were small details in Minecraft which I thought could be improved. I began working, but the project quickly grew out of proportion as I began retexturing the whole game with very low quality textures. This was a deviation from the original goal for me. Realizing this I began a very short lived project called LavidaCraft Favorites. It saved my favorite changes and threw out the rest. Later, I restarted the project as LavidaCraft Official. The word Official was later dropped when I made LavidaCraft 1.5.
